L_NETWORKPROBLEM_FAQ_TEXT_5="Cada dispositivo sem fio na rede sem fio deve ter as mesmas configurações:<br><br><B>SSID</B>: o nome da rede sem fio faz distinção entre letras maiúsculas e minúsculas. (Netgear é diferente de NETGEAR). Se você tiver um roteador predefinido, ele terá um nome de rede sem fio exclusivo, que está em seu rótulo. O SSID padrão para outros roteadores sem fio NETGEAR é NETGEAR.<br><br><B>Canal</B>: os canais sem fio são de 1 a 11 na América do Norte (1 a 13 em outros países).<br><br><B>Modo</B>: o modo sem fio deve ser definido para o modo de infraestrutura para clientes (computadores) para conectar a um roteador sem fio.<br><br><B>Criptografia</B>: você tem essas opções para criptografia:<br><B>.</B>Sem segurança<br><B>.</B>Usar WEP<br><B>.</B>Usar WPA-PSK<br><B>.</B>Usar WPA2-PSK<br><br><B>Rádio sem fio</B>: esta opção deve estar ATIVADA (o LED sem fio do roteador deve estar aceso.)<br><br><B>Lista de controle de acesso (ACL)</B>: se você habilitou uma ACL em seu roteador, certifique-se de que o endereço MAC de cada adaptador sem fio esteja incluído nessa lista."

L_NETWORKPROBLEM_FAQ_LABEL_7="Meu roteador parou de funcionar e o LED de energia pisca lentamente, o que eu devo fazer?"
L_NETWORKPROBLEM_FAQ_TEXT_7="Se o LED de energia do roteador pisca lentamente, significa que o firmware do roteador está corrompido. Você pode usar o seguinte procedimento para recuperar o roteador. Se o procedimento não funcionar e o seu roteador ainda estiver na garantia, entre em contato com o suporte técnico da NETGEAR.<br><br>Etapa 1: Usando uma outra conexão com a Internet (como no trabalho ou em um ponto de acesso sem fio), faça o download do firmware mais recente para o seu roteador em <a href='http://www.netgear.com'>http://www.netgear.com</a>.<br><br>Etapa 2: Use um cabo Ethernet para conectar o seu computador a uma das 4 portas LAN do roteador. Configure o computador com o endereço IP estático 192.168.1.2 e máscara de subrede 255.255.255.0.<br><br><B>.</B>&nbsp;&nbsp;<b>Windows</b>: Selecione <b>Painel de controle &gt; Conexões de rede</b>. Clique com o botão direito do mouse em <b>Conexão local</b> e selecione <b>Propriedades</b>. Em seguida, realce <b>Protocolo de Internet (TCP/IPv4) </b>e clique em <b>Propriedades</b>. Na página de configuração de IP, selecione a caixa de seleção <b>Usar o seguinte endereço IP</b>, digite<b>192.168.1.2</b> para o endereço IP e <b>255.255.255.0</b> para a máscara de sub-rede e clique em <b>OK</b>.<br><B>.</B>&nbsp;&nbsp;<b>Linux ou Mac</b>: Use as instruções do Linux ou Mac para definir o computador com este IP estático.<br><br>Etapa 3: Se ao usar um PC Windows, abra uma janela de comando para Linux ou Mac e use um cliente TFTP.<br><br><B>.</B>&nbsp;&nbsp;<b>Windows 7</b>: Clique no botão <b>Iniciar</b>. Na caixa de pesquisa, digite <b>cmd</b> e pressione <b>Enter</b>.<br><B>.</B>&nbsp;&nbsp;<b>Windows XP</b>: Selecione <b>Iniciar &gt; Executar</b>, digite <b>cmd</b> e pressione <b>Enter</b>.<br><br>Use o comando cd para acessar o diretório (pasta) com o firmware baixado na Etapa 1. Execute o comando <b>tftp -i 192.168.1.1 put firmware_filename</b>.<br><br><B>.</B>&nbsp;&nbsp;<b>Linux ou Mac</b>: Faça o download e execute qualquer cliente TFTP disponível para o seu sistema operacional.<br><br>Etapa 4: Aguarde alguns minutos para que o roteador salve o novo firmware e reinicie. O LED de energia fica verde após uma recuperação bem-sucedida.<br><br>Etapa 5: Defina o endereço IP do computador novamente para IP dinâmico usando um procedimento parecido com o da Etapa 2; selecione a caixa de seleção <b>Obter endereço IP automaticamente</b>."
