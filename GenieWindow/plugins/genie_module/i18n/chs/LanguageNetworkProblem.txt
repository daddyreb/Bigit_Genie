L_NETWORKPROBLEM_FAQ_LABEL_2="我可以连接到路由器，但连不上因特网"
L_NETWORKPROBLEM_FAQ_TEXT_2="如果您可以连接到路由器，但连不上因特网，请登录到路由器，确保因特网服务提供商（ISP）设置正确无误。这些设置位于以下位置的“基本设置”页面中： <a href = 'http://routerlogin.net'>http://routerlogin.net</a>.根据你宽带服务商的信息输入正确的设置。通常为DSL，输入DSL用户名（如：xxx@att.net）和DSL密码。<br><br>如果你不确定使用哪些设置，请联系你的ISP。<br><br>如果已经有使用线缆连接的因特网服务，要确保线缆调制解调器已经关闭并在路由器设置期间一直关闭。如果路由器仍不能从调制解调器上获取因特网连接，可能是因为调制解调器记住了此前设备的MAC地址（例如：在得到路由器前访问因特网的计算机的MAC地址）。你可以在基本设置页输入设备的MAC地址： <a href = 'http://routerlogin.net'>http://routerlogin.net</a>."
L_NETWORKPROBLEM_FAQ_LABEL_3="登录到路由器的密码是什么？"
L_NETWORKPROBLEM_FAQ_TEXT_3="路由器的用户名为 <B>admin</B>。<br><br>默认密码为 <B>password</B>。<br><br>在恢复出厂设置之前，请先尝试默认密码。<br><br>如果您的 NETGEAR 路由器支持<b>密码恢复</b>功能，请在更改路由器密码时启用该功能。<br><br>如果恢复出厂设置，则路由器的所有设置信息都将丢失，您需要全部重新设置一遍！例如，路由器不会记住您的因特网连接设置或者您指定的任何其它自定义设置。<br><br>因此，在恢复出厂设置之前，请准备好您的 ISP 信息。"
L_NETWORKPROBLEM_FAQ_LABEL_4="我的网络应该使用什么类型的无线安全设置？"
L_NETWORKPROBLEM_FAQ_TEXT_4="为了保护您的无线网络，NETGEAR 建议使用 WPA2 或 WPA 无线安全设置（WEP 是一种陈旧的标准，不建议使用）。无线安全设置有助于保护您的网络免受黑客和未授权用户的侵害。<br><br>如果您使用的是已经过预设的路由器，则该路由器会有一个唯一的无线网络名称，并且还设置了 WPA/WPA2 无线安全功能和唯一的密码。购买相同路由器型号的其他人各有针对其网络的不同无线网络名称和密码。<br><br>对于 NETGEAR 路由器，您可以使用 NETGEAR 精灵或智能向导设置 WPA2 或 WPA，也可以在以下位置登录到路由器：<a href = 'http://routerlogin.net'>http://routerlogin.net</a> 并进入无线设置页。当你在路由器设置无线安全，任何人要进行无线连接都必须输入你设置的网络密码或密钥。否则他们不能连接。"
L_NETWORKPROBLEM_FAQ_LABEL_5="对无线网络进行故障诊断"
L_NETWORKPROBLEM_FAQ_TEXT_5="无线网络上的各个无线设备必须具有相同的设置：<br><br><B>SSID</B>：无线网络名称区分大小写（Netgear 不同于 NETGEAR）。如果您使用的是已经过预设的路由器，则该路由器会有一个唯一的无线网络名称，它在路由器标签上。其它 NETGEAR 无线路由器的默认 SSID 为 NETGEAR。<br><br><B>频道</B>：在北美洲，无线频道为 1 到 11（其它国家或地区为 1 到 13）。<br><br><B>模式</B>：要使客户端（计算机）连接到无线路由器，无线模式应设置为基础结构模式。<br><br><B>加密</B>：您可以使用以下加密选项：<br><B>.</B> 无安全设置<br><B>.</B> 使用 WEP<br><B>.</B> 使用 WPA-PSK<br><B>.</B> 使用 WPA2-PSK<br><br><B>无线收发功能</B>：该设置应为“打开”（路由器的无线 LED 应点亮）。<br><br><B>访问控制列表 (ACL)</B>：如果您在路由器上启用了 ACL，请确保将各个无线适配器的 MAC 地址包含在该列表中。"

L_NETWORKPROBLEM_FAQ_LABEL_7="我的路由器停止工作，并且电源 LED 指示灯缓慢闪烁，我应该怎么办？"
L_NETWORKPROBLEM_FAQ_TEXT_7="如果路由器的电源 LED 指示灯缓慢闪烁，表示路由器的固件已损坏。您可以使用以下过程来恢复路由器。如果此过程不起作用，并且您的路由器仍在保修期内，请联系 NETGEAR 技术支持部门。<br><br>第一步：使用其他因特网连接（如在办公室或 Wi-Fi 热点），从 <a href = 'http://www.netgear.com'>http://www.netgear.com</a> 下载适用于您路由器的最新固件。<br><br>第二步：使用以太网电缆将计算机连接到路由器上 4 个 LAN 端口中的一个。将计算机配置为使用静态 IP 地址 192.168.1.2 和子网掩码 255.255.255.0。<br><br><B>.</B>&nbsp;&nbsp;<b>Windows</b>：选择<b>控制面板 > 网络连接</b>。右键单击<b>本地连接</b>并选择<b>属性</b>，然后突出显示 <b>Internet 协议 (TCP/IPv4)</b>，再单击<b>属性</b>。在 IP 配置页面中，选中<b>使用下面的 IP 地址</b>复选框，输入 <b>192.168.1.2</b> 作为 IP 地址，并输入 <b>255.255.255.0</b> 作为子网掩码，然后单击<b>确定</b>。<br><B>.</B>&nbsp;&nbsp;<b>Linux 或 Mac</b>：使用 Linux 或 Mac 说明来将计算机设置为使用此静态 IP。<br><br>第三步：如果使用 Windows PC，请打开命令窗口；对于 Linux 或 Mac，则使用 TFTP 客户端。<br><br><B>.</B>&nbsp;&nbsp;<b>Windows 7</b>：单击<b>开始</b>按钮。在“搜索”框中，输入 <b>cmd</b> 并按 <b>Enter</b> 键。<br><B>.</B>&nbsp;&nbsp;<b>Windows XP</b>：选择<b>开始 > 运行</b>，输入 <b>cmd</b> 并按 <b>Enter</b> 键。<br><br>使用 cd 命令转至包含您在第一步中下载的固件的目录（文件夹）。运行命令 <b>tftp -i 192.168.1.1 put firmware_filename</b>。<br><br><B>.</B>&nbsp;&nbsp;<b>Linux 或 Mac</b>：下载并运行可用于您操作系统的任何 TFTP 客户端。<br><br>第四步：等待数分钟，使路由器保存新固件，然后重启。成功恢复后，电源 LED 指示灯将变为绿色。<br><br>第五步：使用类似于第二步的过程，选中<b>自动获得 IP 地址</b>复选框，从而将计算机的 IP 地址重新设置为动态 IP。"
